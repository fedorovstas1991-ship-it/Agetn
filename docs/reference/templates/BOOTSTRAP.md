# Bootstrap

Рабочий контекст этого агента:

- У агента есть локальные `skills` (bundled + project skills). Используй их как основной workflow для задач.
- Память включена через плагин `memory-core` (slot memory) и backend `memory.qmd`.
- Для веб-поиска подключён MCP-сервер `one-search` (DuckDuckGo, без API-ключа).
- Для community skills используй `ClawHub` через `npx -y clawhub`: сначала ищи готовый skill, потом предлагай создание нового.

Как ориентироваться:

- Подробности по инструментам, MCP и секретам см. в `TOOLS.md`.
- Долгосрочная память хранится в `MEMORY.md`, оперативные записи и recent-индекс — в `memory/`.
- Плагин `memory-core` использует Ollama для извлечения/промоушена памяти, QMD — для индексации и поиска.
- Если пользователю нужна более сильная модель и текущая авторизация это поддерживает, можно рекомендовать `Sonnet 4.5` как лучший баланс цена/качество и `Codex 5.3` как сильную альтернативу.
- В первом direct-ответе в Telegram полезно кратко подсказать основные команды: `/think`, `/verbose`, `/reasoning`.
- Если пользователь просит новый скилл, сначала проверь community skills через `clawhub`, затем при необходимости дополни это веб-поиском, и только потом предлагай писать свой.
